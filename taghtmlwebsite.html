Asal usul, pengertian dan 50 tag HTML

50 Tag HTML dan fungsinya

Nah sebelum kita membahas tag-nya, kalian tau gak sih asal usul HTML? Mari kita bahas.


   HTML (HyperText Markup Language) ditemukan oleh Tim Berners-Lee, seorang ilmuwan komputer asal Inggris, pada tahun 1989. Tim Berners-Lee bekerja di CERN (European Organization for Nuclear Research) di Swiss ketika ia mengembangkan HTML sebagai bagian dari konsep World Wide Web (WWW).

Versi Awal HTML

Versi pertama HTML (HTML 1.0) diluncurkan pada tahun 1993. Versi ini sederhana dan hanya mendukung elemen-elemen dasar seperti:
  • Teks (judul, paragraf)
  • Tautan (hyperlink)
  • Gambar
  • Daftar

HTML Hari ini

  • HTML 5 adalah versi yang paling populer dan digunakan saat ini.
  • HTML kini menjadi tulang punggung semua situs web, memungkinkan pengembang untuk menciptakan pengalaman pengguna yang interaktif dan kaya.

HTML bukan bahasa pemrograman?

Walaupun sering disebut sebagai bahasa pemrograman, HTML sebenarnya adalah bahasa markup. Artinya, HTML digunakan untuk menandai (markup) konten agar bisa ditampilkan di web browser.

HTML adalah fondasi web:

Semua halaman web yang kamu lihat dibangun di atas kerangka HTML. HTML menentukan struktur dasar suatu halaman, seperti judul, paragraf, gambar, dan tautan.

HTML terus berkembang:

HTML tidak statis. Versi terbaru, HTML5, memiliki banyak fitur baru yang memungkinkan pembuatan halaman web yang lebih dinamis dan interaktif.

HTML bekerja dengan bahasa lain:

HTML sering dikombinasikan dengan CSS untuk mengatur tampilan (styling) dan JavaScript untuk menambahkan interaktivitas pada halaman web.

Berikut 50 tag HTMl beserta fungsinya


  • <html>: Tag utama yang digunakan untuk mendefinisikan dokumen HTML.
  • <head>: Berisi informasi meta, link, script, dan judul dokumen.
  • <title>: Menampilkan judul halaman di tab browser.
  • <meta>: Menyediakan metadata tentang dokumen seperti karakter set, deskripsi, dan lainnya.
  • <link>: Digunakan untuk menghubungkan dokumen HTML dengan file eksternal (misalnya CSS).
  • <body>: Menyimpan konten yang akan ditampilkan di halaman web.
  • <h1> hingga <h6>: Tag untuk judul dengan level hierarki, <h1> adalah yang paling besar.
  • <p>: Digunakan untuk menulis paragraf.
  • <a>: Digunakan untuk membuat tautan (hyperlink).
  • <img>: Digunakan untuk menyisipkan gambar ke dalam halaman.
  • <ul>: Membuat daftar tidak berurutan (unordered list).
  • <ol>: Membuat daftar berurutan (ordered list).
  • <li>: Menandakan elemen dalam daftar (list item).
  • <div>: Membagi halaman menjadi blok, digunakan untuk struktur layout.
  • <span>: Membungkus teks dalam elemen inline untuk styling atau manipulasi.
  • <table>: Membuat tabel.
  • <tr>: Menandakan baris dalam tabel.
  • <th>: Menandakan header kolom dalam tabel.
  • <td>: Menandakan sel data dalam tabel.
  • <form>: Digunakan untuk membuat form input data.
  • <input>: Elemen untuk berbagai jenis input (misalnya teks, checkbox, radio button).
  • <button>: Menampilkan tombol yang bisa diklik.
  • <label>: Memberikan label untuk elemen form input.
  • <select>: Menyediakan dropdown list.
  • <option>: Menyediakan opsi dalam elemen <select>.
  • <textarea>: Membuat area teks multi-line (textarea) dalam form.
  • <br>: Membuat pemisah baris, berguna untuk line break.
  • <hr>: Membuat garis horizontal untuk pemisahan konten.
  • <strong>: Menyatakan teks yang ditekankan atau dicetak tebal.
  • <em>: Menyatakan teks yang dicetak miring untuk penekanan.
  • <q>: Digunakan untuk menandai kutipan pendek.
  • <cite>: Menyatakan referensi atau sumber dari kutipan.
  • <code>: Menandai teks sebagai kode program.
  • <kbd>: Menandai input dari keyboard.
  • <samp>: Menandai hasil output dari program.
  • <var>: Menandai variabel dalam kode pemrograman.
  • <mark>: Menandai teks yang relevan atau disorot.
  • <progress>: Menampilkan indikator progres atau kemajuan.
  • <meter>: Menampilkan skala atau ukuran dalam rentang tertentu.
  • <svg>: Menyisipkan grafis vektor berbasis XML dalam dokumen HTML.
  • <canvas>: Menyediakan area untuk menggambar grafis dinamis menggunakan JavaScript.
  • <audio>: Menyisipkan elemen audio dalam halaman.
  • <video>: Menyisipkan elemen video dalam halaman.
  • <iframe>: Menyisipkan halaman HTML lain ke dalam halaman saat ini.
  • <script>: Digunakan untuk menyisipkan atau merujuk ke file JavaScript.
